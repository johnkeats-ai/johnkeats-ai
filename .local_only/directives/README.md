# Directives

Directives are Standard Operating Procedures (SOPs) written in Markdown. They define the goals, inputs, tools, and outputs for specific tasks.

## Structure
- **Goal**: What are we trying to achieve?
- **Inputs**: What data is needed?
- **Tools**: Which scripts in `execution/` should be used?
- **Outputs**: What are the expected deliverables?
- **Edge Cases**: What should be handled if things go wrong?
